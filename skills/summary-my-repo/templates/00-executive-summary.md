# 00 Overview Template

Write to:

- `summary-my-repo/YYYY-MM-DD-vN/00-overview.md`

## Suggested Sections

1. What this repository is
2. Why the architecture is shaped this way
3. Top-level architecture snapshot
4. Main workflows
5. Critical invariants
6. Risks or current gaps
7. Recommended reading order

## Rule

Make this the entry document for a new collaborator. It should be readable without opening any other file first.
For core behavior claims, reference snippet ids from `03-code-evidence.md` inline.
