# 01 Directory Map Template

Write to:

- `summary-my-repo/YYYY-MM-DD-vN/01-directory-map.md`

## Suggested Sections

1. Top-level tree
2. Responsibility of each top-level directory
3. Important subdirectories and why they exist
4. Which directories are source of truth versus generated output
5. Where a maintainer should add new work

## Rule

Do not just paste a tree. Pair each node with its responsibility and maintenance implications.
