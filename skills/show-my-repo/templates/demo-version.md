# Demo Version Template

## Structure

1. Open with the pain in one sentence
2. Give the one-line answer
3. Show a 3-step demo path
4. Call out 2 visible highlights
5. Add 1 short technical credibility point
6. Close with the next product or business direction

## Length

180 to 300 words.

## Rule

Lead with what the audience can see happen, not internal implementation details.
