# Devtool Example

## Raw Fact Pattern

- CLI tool with clear install flow
- Parses repositories and generates dependency maps
- Includes tests and a GitHub Action

## Good Framing

- User value: helps engineering leads understand dependency risk faster than manual graph tracing
- Technical credibility: parser, graph builder, and report layer are separated cleanly
- Execution signal: test coverage and CI make it look maintained rather than throwaway

## Avoid

- Calling it a platform if it is still a single-purpose CLI
- Claiming market leadership or moat without evidence
