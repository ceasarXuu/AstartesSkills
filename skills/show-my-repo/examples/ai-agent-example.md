# AI Agent Example

## Raw Fact Pattern

- Multi-step task orchestration
- Tool invocation layer
- Local state plus resumable runs
- Prompt templates and evaluation scripts

## Good Framing

- User value: turns fragmented, manual multi-tool workflows into a guided task chain
- Technical credibility: orchestration, persistence, and evaluation suggest a real attempt at reliability
- Commercial framing: useful where complex tasks repeat, but distribution and trust remain major hurdles

## Avoid

- Claiming proprietary intelligence if the main capability comes from external models
- Equating a long prompt library with durable defensibility
